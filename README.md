# ✨ امکانات پروژه

## 👤 سیستم احراز هویت

- ثبت‌نام کاربران
- ورود به حساب کاربری
- جلوگیری از ثبت‌نام با نام کاربری تکراری
- پشتیبانی از دو نوع حساب:
  - 🎤 هنرمند (Artist)
  - 🎧 شنونده (Listener)
- بارگذاری تصویر پروفایل
- ویرایش اطلاعات حساب کاربری
- تغییر رمز عبور
- حذف حساب کاربری

---

## 🎤 پنل هنرمند

هنرمندان می‌توانند به طور کامل آثار موسیقی خود را مدیریت کنند.

### مدیریت آلبوم

- ایجاد آلبوم جدید
- ویرایش اطلاعات آلبوم
- حذف آلبوم
- بارگذاری تصویر کاور آلبوم

### مدیریت آهنگ

- افزودن آهنگ
- ویرایش اطلاعات آهنگ
- حذف آهنگ
- قرار دادن آهنگ در آلبوم
- پشتیبانی از بخش **Singles** برای آهنگ‌های بدون آلبوم

### مدیریت خودکار داده‌ها

- حذف یک آلبوم، تمام آهنگ‌های آن را نیز حذف می‌کند.
- حذف یک آهنگ، آن را از تمامی پلی‌لیست‌های کاربران حذف خواهد کرد.

---

## 🎧 پنل شنونده

کاربران می‌توانند موسیقی را جستجو، مدیریت و پخش کنند.

### مدیریت پلی‌لیست

- ایجاد پلی‌لیست
- ویرایش پلی‌لیست
- حذف پلی‌لیست

### کاوش موسیقی

- مشاهده هنرمندان
- مشاهده آلبوم‌ها
- افزودن آهنگ‌ها به پلی‌لیست شخصی

### علاقه‌مندی‌ها

- افزودن آهنگ به لیست علاقه‌مندی‌ها
- حذف از علاقه‌مندی‌ها
- نمایش آهنگ‌های مورد علاقه در یک پلی‌لیست اختصاصی

### جستجو و فیلتر

امکان جستجوی آهنگ‌ها بر اساس:

- نام آهنگ
- سبک موسیقی
- سال انتشار

### پخش موسیقی

پخش فایل‌های صوتی با استفاده از **Qt Multimedia**

فرمت‌های پشتیبانی‌شده:

- MP3
- WAV

# 🏗️ معماری پروژه

در طراحی این پروژه از اصول مهندسی نرم‌افزار استفاده شده است.

## معماری MVC

| لایه | وظیفه |
|------|--------|
| **Model** | موجودیت‌ها و Repositoryها |
| **View** | رابط کاربری طراحی‌شده با Qt Designer |
| **Controller** | مدیریت ارتباط بین رابط کاربری و داده‌ها |

---

## الگوی Repository

ساختار Repositoryها به صورت زیر طراحی شده است:

```text
AbstractRepository<T>
        │
        ▼
AccountRepository
        │
   ┌────┴────┐
   ▼         ▼
ArtistRepository
ListenerRepository
```

تمام عملیات مربوط به ایجاد، ویرایش، حذف و جستجوی اطلاعات (CRUD) از طریق این لایه انجام می‌شود.

تمام اطلاعات برنامه به صورت دائمی در فایل‌های **JSON** ذخیره می‌شوند.

ویژگی‌ها:

- تبدیل خودکار اشیاء به JSON
- بازیابی اطلاعات از JSON
- ذخیره خودکار تغییرات
- ذخیره تصاویر به صورت Base64

# 🚀 نحوه اجرای پروژه

## پیش‌نیازها

- Qt 6.x
- کامپایلر MinGW یا MSVC
- استاندارد C++17

## اجرای پروژه

1. پروژه را در Qt Creator باز کنید.
2. گزینه **Build → Rebuild All** را اجرا کنید.
3. برنامه را اجرا (Run) نمایید.

---

## ساخت فایل اجرایی مستقل

برای ساخت نسخه مستقل برنامه در ویندوز:

